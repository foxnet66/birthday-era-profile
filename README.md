# Birthday Era Profile

一个静态网页应用，用出生日期生成「生日时代画像」报告。

项目通过出生年份、季节、月内位置、年度位置、日期行为原型和生命主题数，生成一份偏心理画像和代际语境分析的报告。

## 功能

- 输入出生日期，按年 / 月 / 日顺序填写。
- 生成 7 个分析模块：
  - 童年背景解码器
  - 性格进化地图
  - 职业指南针
  - 关系模式分析
  - 财务心态矩阵
  - 隐藏困境探测器
  - 人生路线图
- 生成专属画像摘要，展示代际底盘和个人化变量。
- 一键导出独立 HTML 报告。
- 纯前端实现，不依赖后端服务，不上传生日数据。

## 使用

直接用浏览器打开：

```text
index.html
```

如果需要本地预览服务器：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173/index.html
```

## 分析口径

报告由以下层级共同生成：

- 代际底盘：出生年份对应的时代背景和社会文化环境。
- 出生季节：春、夏、秋、冬对应的成长节奏。
- 月内位置：月初、月中、月末对应的行为倾向。
- 年度位置：年初、年中、年末出生对应的同届节奏差异。
- 日期行为原型：根据具体日期生成个人化表达。
- 生命主题数：基于出生日期数字生成叙事主题。

## 合规说明

本项目定位为启发式画像和代际分析工具，不提供确定性判断、医疗建议、心理诊断、投资建议或法律建议。
